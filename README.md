# charge-server

an Open Charge Point Protocol (OCPP)1.6 java charge server,including development design documents.

充电桩OCPP服务端，实现OCPP 1.6协议解析，完成充电及计费流程，包括详细需求及设计文档

相关知识：
* springboot
* Webservice
* MQTT
* Ccframe基础框架
